# Novixa Connect Pack

**Package code:** `novixa_connect`  
**Brief:** [docs/novixa/03-solution/novixa-connect-pack-brief-v1.md](../../../../docs/novixa/03-solution/novixa-connect-pack-brief-v1.md)

Healthcare **collaboration** workspace — referral, booking, org network.  
Does **not** issue prescriptions or perform clinical care.

Pharmacy and Clinic packs remain separate. Connect orchestrates between them.
